# [백준 Java 8 (OpenJDK) 기준](https://help.acmicpc.net/language/info)

1. 클래스 명: Main
2. 시간 제한: x2 + 1초
3. 메모리 제한: x2 + 16MB
4. 컴파일 명령: javac -J-Xms1024m -J-Xmx1920m -J-Xss512m -encoding UTF-8 Main.java
5. 실행 명령: java -Xms1024m -Xmx1920m -Xss512m -Dfile.encoding=UTF-8 Main
